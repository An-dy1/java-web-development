4.1. Java Lesson 4 Objectives¶

4.1.1. Goals¶

  1. Use modifiers to customize fields and methods
  2. Override built-in methods such as toString and equals
  3. Understand equality and sameness in Java

4.1.2. Objectives¶

  1. Explain and properly use the static keyword for fields and methods
  2. Explain and properly use the final keyword
  3. Create constant fields using static final
  4. Explain why instance variables may not be referenced within static methods
  5. Write a custom toString method for a given class
  6. Explain the concepts of object identity and equality, and how they differ
  7. Explain and properly use .equals to determine when two objects are equal
  8. Explain the behavior of == with respect to objects in Java
  9. Describe and implement the best practices for implementing .equals within a class
  10. Explain when .hashCode should be implemented within a class
  11. Use IntelliJ’s code generation tools to generate well-structured .equals and .hashCode methods

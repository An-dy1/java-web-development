8.2. Exercises: Interfaces and Polymorphism¶

We’ve learned that ArrayList<T> implements both List<E> and Iterable<T>. In fact, List<E> extends Iterable<T>, which means that List<E> includes all of the contracted methods that Iterable<T> does.

Check out the documentation for List<E> and Iterable<T> to confirm for yourself!
